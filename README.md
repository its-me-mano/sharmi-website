# sharmi-website

A personal website project — a birthday surprise for Sharmila. 🎉

## Contents

- `Happy Birthday Sharmila.html` — the main birthday page.

## Running locally

This is a static site. Open the HTML file directly in a browser, or serve the folder:

```bash
# Python 3
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Deploying

Any static host works (GitHub Pages, Netlify, Vercel). For GitHub Pages, enable
Pages on the `main` branch in the repository settings.
